## Tech Stack

* **React (JavaScript)**
* **Vite**
* **React Router DOM**
* **Tailwind CSS**
* **JSON Server**
* **React Toastify**

## Installation

### Step 1: Clone the Repository

```bash
git clone https://github.com/Hardik0602/management-app
cd management-app
```

### Step 2: Install Dependencies

```bash
npm install
```

### Step 3: Start the Local Backend

```bash
npm run server
```

### Step 4: Start the Development Server

```bash
npm run dev
```
